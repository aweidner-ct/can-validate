can-validate.validator  

Registers a library with can.validate. The last one registered is the default library. Override the default by changing _validatorId to the key of the desired registered library.

Validate.register('validatejs',validatejs);

utils.validator(id, validator)

Registers a library with can.validate. The last one registered is the default library. Override the default by changing _validatorId to the key of the desired registered library.

Validate.register('validatejs',validatejs);

Parameters

  1. id {string}

    The key name of the validator library.

  2. validator {object | function()}

    The validator libarary. Only pass instances.
