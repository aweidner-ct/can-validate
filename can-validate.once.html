can-validate.once  

Registers a library with can.validate. The last one registered is the default library. Override the default by changing _validatorId to the key of the desired registered library.

Validate.once('', {}, 'myVal');

validators.once(value, options, name)

Registers a library with can.validate. The last one registered is the default library. Override the default by changing _validatorId to the key of the desired registered library.

Validate.once('', {}, 'myVal');

Parameters

  1. value {*}

    A value of any type to validate.

  2. options {object}

    Validation config object, structure depends on the validation library used.

  3. name {string}

    Optional. The key name of the value.
