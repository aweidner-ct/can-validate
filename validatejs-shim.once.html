validatejs-shim.once  

Validates a single property using provided validation options

validatejs-shim.once(value, options, name)

Validates a single property using provided validation options

Parameters

  1. value {*}

    Some value to validate against.

  2. options {Object}

    Raw validation options. They will be processed since not all options are valid for ValidateJS.

  3. name {string}

    The key name of the value to validate. Used to prepend to error messages, if any.

Returns

{undefined | array}

Returns undefined if no errors, otherwise returns a list of errors.
