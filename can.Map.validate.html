Map Plugin

  • module
can.Map.validate  

The can.Map plugin will works alongside can.Map.define to add validation to properties on a can.Map. Importing the plugin, validation library, and a shim will allow the ability to dynamically check values against validation configuration. errors are stored on the can.Map instance and are observable.

Initialization

Import the validation library, validate plugin and a shim to immediately use the can.Map.validate plugin.

import 'validatejs';
import 'can-validate/map/validate';
import 'can-validate/shims/validatejs.shim';
